ADA Transformer System

PHP JavaScript jquery HTML/CSS Bootstrap Mongo DB Apex Charts MQTT JSON

The Advanced Distribution Analytics â€“ Transformer System (ADA-TS) is an innovative platform designed to enhance the monitoring, management, and reliability of power distribution transformers. This intelligent system leverages real-time data analytics, sensors, and advanced algorithms to assess transformer performance and overall health. ADA-TS enables utility providers to track critical parameters such as load levels, tank and ambient temperature, current, and fault indicators, ensuring early detection of potential issues. The system also features an alarm function to provide timely alerts.

Key Features:

  • Homepage (Real-Time Monitoring Dashboard)
    • Displays a real-time interactive map of all installed transformers.
    • Displays a real-time interactive map of all installed transformers.
    • Transformer location pins change color or icon based on current device condition:
      • Normal
      • Underload
      • Overload
      • Critical
      • Outage
      • Communication Issue
    • Shows Total Installations, Total Devices Installed, and Total Power Saved across all devices.
    • Displays Population per kVA to analyze transformer load per area.
    • Includes Installation Statistics to track growth and deployment coverage.
    • Provides a search function to find transformers by TLN (Transformer Line Number) or Model.
    • Clicking a transformer pin opens a detailed modal with complete device specifications, real-time data, and status history.
  • Dashboard
    • Provides real-time visualization of the current status and operational data of a selected transformer.
    • Includes date range filters to view and analyze historical performance, events, and fault data.
    • Features a Data Correlation tool with 4 dropdown menus to compare and analyze multiple parameters side by side.
    • Includes a dynamic data table with column visibility options, allowing users to display or hide specific parameters for easier analysis.
  • Real-Time Alerts
    • Sends instant email notifications and texts to users when a transformer reaches a critical status or fault condition.
    • Maintains a comprehensive alerts data table that logs:
      • Date and time of the alert
      • Device description
      • Location of the transformer
      • Alert details and status
  • User Roles & Access Control
    • Super Admin:
      • Full control over the system such as all the devices installed per company.
      • Can create, update, and delete admin accounts.
      • Has access to all system functionalitiesa and user logs.
    • Admin:
      • Can manage and monitor data for a specific company.
      • Can generate reports and monitor asset status.
      • Receives notification alerts from devices.
      • Has limited access compared to the Super Admin but can add users under the same company.
    • Users:
      • Can monitor the data in specific company.
      • Generate reports and monitor asset status.
  • Reporting & Analytics
    • Generate real-time reports on events, parameters table and graphs.
    • Export reports in PDF or Excel format.
    • Visual dashboards displaying real time data using Apex Charts.
  • System Security & Data Integrity
    • User authentication with role-based access.
    • Database backup and recovery feature.
    • Logs for tracking changes made to inventory records.
  • Technology Stack
    • Frontend: HTML, CSS, JavaScript (with AJAX & jQuery for interactivity).
    • Backend: PHP (Object-Oriented Programming for maintainability).
    • Database: NoSQL - MongoDB (Structured asset management with relational tables).
    • Communication Protocol: MQTT (used for reliable real-time data transmission between devices and the system).
    • Data Format: JSON (data from devices is transmitted in JSON format for efficient storage and processing).
    • Security: SSL encryption and role-based authentication.

A powerful platform designed to help electric utilities monitor, manage, and analyze transformer health and performance in real time. By integrating devices, MQTT communication, and JSON-based data processing, the system delivers accurate insights, predictive maintenance alerts, and comprehensive reporting to improve operational efficiency. Its intelligent visualization tools, automated notifications, and detailed analytics empower utilities to prevent outages, optimize asset performance, and ensure reliable power distribution.
